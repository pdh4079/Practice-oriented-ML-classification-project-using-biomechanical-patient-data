# Biomechanical Features 분류 프로젝트 (KNN)

## 프로젝트 개요
본 프로젝트는 **Biomechanical Features of Orthopedic Patients** 데이터셋을 활용한  
학습 목적의 머신러닝 분류 프로젝트입니다.

정형 데이터 기반 분류 문제를 통해 머신러닝 분석의 기본 흐름  
(데이터 탐색 → 모델 선택 → 파라미터 실험 → 성능 평가)을 이해하는 것을 목표로 진행했습니다.

본 레포지토리는 튜토리얼 기반으로 시작하였으나,  
단순 구현에 그치지 않고 **직접 이해하고 설명 가능한 범위의 분석만 정리**하는 데 초점을 두었습니다.

---

## 데이터셋
- **출처**: Kaggle  
- **설명**: 정형외과 환자의 생체역학적 지표를 기반으로 한 분류 데이터  
- **유형**: 정형(Structured) 의료 데이터 / 이진 분류 문제

---

## 접근 방법
- 데이터의 기본 분포와 특성을 확인하기 위한 탐색적 데이터 분석(EDA) 수행
- 거리 기반 분류 모델인 **KNN (k-Nearest Neighbors)**을 기준 모델로 선택
- 이웃 수(K) 변화에 따른 성능 변화를 실험하며 모델 특성 이해

---

## 성능 평가
- **Accuracy, Precision, Recall** 지표를 사용하여 모델 성능 평가
- 실험 결과, **K = 13**에서 가장 안정적인 성능을 확인
- 단일 지표가 아닌, 클래스별 성능 지표를 함께 고려하는 평가의 중요성을 인식

---

## 주요 인사이트
- 거리 기반 모델은 데이터 분포 및 특성 스케일에 큰 영향을 받음
- Accuracy만으로는 분류 성능을 충분히 설명하기 어렵고,  
  Precision과 Recall을 함께 해석하는 것이 중요함
- 성능 수치 자체보다 **왜 해당 결과가 나왔는지를 해석하는 과정이 핵심**임을 학습
